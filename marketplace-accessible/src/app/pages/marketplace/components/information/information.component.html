¿Quines somos? MARKETPLACE es un sitio web para vender y comprar diversos artículos, desde un videojuego para tu consola, hasta el nuevo auto de tus sueños, aunque esto no estaría completo sin una interfaz y detalles accesibles para usuarios con previa experiencia en sitios web tipo Marketplace y personas con TEA. ¿Que es TEA? El Trastornos del Espectro Autista (TEA) es una discapacidad del desarrollo que puede provocar problemas sociales, comunicacionales y conductuales significativos. Dentro de un sitio web tipo Marketplace pueden presentar problemas a la hora de obtener la información de un producto, comunicarse con el vendedor o determinar que acciones esta realizando en cada momento (e.j: cuando está publicando un artículo para vender). Es por ello que es importante que cada vez que des información sobre algún producto seas claro a la hora de detallar a que se esta refiriendo, para evitar confusiones que lleven a la no compra del artículo. ¿Que significa cada color en nuestra página?

 ¿Que buscamos? Como equipo buscamos generar una nueva forma de comprar y vender artículos en un sitio web, en donde nadie se sienta discriminado y pueda satisfacer sus necesidades fácilmente, donde la información que se lea dentro del sitio web sea sencilla de entender y agradable visualmente, sin imágenes o iconos que puedan confundir al usuario. Además, buscamos sumar a todos los usuarios de este sitio web a comprender las necesidades de otras personas y comprender que todos merecemos una buena experiencia al comprar y vender artículos por internet.
